🚀 Deploy on Vercel (Step by Step)
Option A — Upload via Vercel Dashboard (Easiest, no coding)
Create a free account at vercel.com (sign up with Google)
Click "Add New Project" → choose "Import from file / folder" or use Vercel CLI
Or use the drag-and-drop method:
Go to vercel.com/new
Scroll down → click "Deploy from your computer"
Drag your entire krishnaveni-agro/ folder into the box
Click Deploy
✅ Your site is live! Vercel gives you a free URL like krishnaveni-agro.vercel.app
Option B — GitHub + Vercel (Best for updates)
Create a GitHub account at github.com
Create a new repository called krishnaveni-agro
Upload all your files (index.html, style.css, script.js, images/ folder)
Go to vercel.com → "Add New Project" → "Import Git Repository"
Select your GitHub repo → click Deploy
✅ Every time you update files on GitHub, the website updates automatically!
Option C — Vercel CLI (For developers)
bash
npm install -g vercel
cd krishnaveni-agro
vercel
Follow the prompts. Done in under 1 minute.

📸 How to Add Your Own Photos
Take photos of your farm, family, dragonfruit
Name them exactly as listed in the folder structure above
Put them in the images/ folder
Re-deploy (or push to GitHub if using Option B)
Recommended photo sizes:

logo.png — 200×200px (square)
founder.jpg, gallery-1.jpg — 800×600px or larger
family-*.jpg, health-*.jpg — 400×400px (square crops best)
ig-*.jpg — 600×600px (square)
product-main.jpg — 800×800px
🌐 Custom Domain (Optional)
Buy a domain (e.g., krishnaveniagro.com) from GoDaddy / Namecheap (~₹800/year)
In Vercel dashboard → your project → "Domains" → Add your domain
Follow the DNS instructions Vercel gives you
Done! Your site runs on your own domain.
